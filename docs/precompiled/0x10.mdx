## Notes

BLS12-381 map field element to G1 precompile maps base field elements (Fp) into G1 points. This operation is required for signature schemes and performs field arithmetic to map a field element into a curve point.

This precompile does NOT perform mapping of byte strings into field elements - it only handles the field-to-point mapping. The mapping uses the Simplified Weighted Sum (SWU) method with parameters specified in EIP-2537. More information can be found in [EIP-2537](https://eips.ethereum.org/EIPS/eip-2537).

## Inputs

| Byte range | Name | Description |
|-----------:|-----:|------------:|
| `[0; 63]` (64 bytes) | field_element | Base field element (Fp) to map to G1 |

Input must be exactly 64 bytes. The field element must be a valid element of the BLS12-381 base field (less than the field modulus).

## Output

| Byte range | Name | Description |
|-----------:|-----:|------------:|
| `[0; 63]` (64 bytes) | x | X coordinate of mapped G1 point |
| `[64; 127]` (64 bytes) | y | Y coordinate of mapped G1 point |

Returns a G1 point that is the deterministic mapping of the input field element. The output point is guaranteed to be in the correct G1 subgroup. Gas cost: **5,500 gas**. If the input is not valid, all gas provided is consumed and there is no return data, indicating a precompile contract error.
